# Security Policy

## Reporting security issues

Please do not open public GitHub issues for security-sensitive reports.

Instead, contact the repository maintainer privately using the contact
method listed on the GitHub profile.

## Scope

This integration observes local HTTP traffic from a Level Sense Pro and
forwards it to the vendor cloud. It does not require vendor credentials
and does not intentionally store secrets.

Diagnostics may include local network details such as IP addresses, MAC
addresses, and device identifiers. Review diagnostics before posting
them publicly.

## Supported Versions

The current release branch receives fixes and documentation updates.
When reporting a security issue, include the integration version and
Home Assistant version if known.
